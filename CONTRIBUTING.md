# Contributing to HuggingFace Colab Secure Setup

Thank you for your interest in contributing! This project aims to make AI/ML accessible with privacy and security.

## How to Contribute

### Reporting Issues

- Check existing issues before creating new ones
- Use clear titles and detailed descriptions
- Include reproduction steps and environment info
- Tag appropriately (bug, feature, question)

### Submitting Changes

1. **Fork the repository**
2. **Create a feature branch**
   ```bash
   git checkout -b feature/your-feature-name
   ```
3. **Make your changes**
   - Follow existing code style
   - Add tests if applicable
   - Update documentation
4. **Commit with clear messages**
   ```bash
   git commit -m "Add: brief description of changes"
   ```
5. **Push and create PR**
   ```bash
   git push origin feature/your-feature-name
   ```

### Code Style

- Use clear variable names
- Add comments for complex logic
- Follow PEP 8 guidelines
- Include type hints where helpful

### Notebook Guidelines

- Clear cell outputs before committing
- Include markdown explanations
- Test notebooks end-to-end
- Keep dependencies minimal

## Types of Contributions

### Bug Fixes
- Include test cases
- Update related documentation
- Reference issue number in commit

### New Features
- Discuss major changes in issues first
- Provide working examples
- Add appropriate tests

### Documentation
- Fix typos and improve clarity
- Add missing examples
- Keep docs up-to-date with code

### Security Contributions
- Follow security best practices
- Report vulnerabilities responsibly
- Help improve privacy features

## Questions?

Feel free to open an issue for questions or discussion.
